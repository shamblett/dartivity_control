About

Dartivity Control is a web based front end to the Dartivity IOT management package.

Dartivity is a server based Dart package that combines various IOT client implementations such as Iotivity with messaging systems such as Google's pub/sub service and MQTT to provide a distributed decoupled IOT device management and control infrastructure.

For further information and the current facilities provided by Dartivity and Dartivity Control please refer to the documentation in the github repositories, here and here
